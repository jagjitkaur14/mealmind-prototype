# GitHub Update Steps for mealmind-prototype

## 1. Replace README.md

1. Open the repository:
   https://github.com/jagjitkaur14/mealmind-prototype
2. Click `README.md`.
3. Click the pencil/edit icon.
4. Delete the current one-line content.
5. Paste the new README content from this folder.
6. Commit the change with this message:

```text
Update README with product case-study documentation
```

## 2. Update GitHub About Description

Use this description:

```text
A smart meal-planning prototype that generates weekly meal plans using user preferences, pantry inventory, cooking time, and an auto-created shopping list.
```
